# The Daily News Point

The future of information. A premium viral news website built with Next.js, Tailwind CSS 4, and Framer Motion.

## Features

- High-contrast "Modern Noir" design
- Classic Newspaper Masthead with separate Sticky Navigation
- Framer Motion animations
- Fully responsive layout
- Black and White aesthetic

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Icons**: Lucide React
- **Animation**: Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
